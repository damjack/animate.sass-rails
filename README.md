# animate.sass-rails

Make animate.sass easily usable in your Rails application.

## Setup

1) Add to your `Gemfile`:

```
gem 'animate.sass-rails'
```

2) Run `bundle install` or simply `bundle`.

3) In you `application.sass` or whatever SASS file of your app:

```
@import 'animate'
```

4) Enjoy.
